# Customer Churn Prediction using Machine Learning and Deep Learning

## 📌 Overview

Customer churn prediction is a classification problem that helps businesses identify customers who are likely to stop using their services. This project compares traditional Machine Learning algorithms with Deep Learning models to accurately predict customer churn and evaluate their performance.

## 🎯 Objectives

- Predict whether a customer will churn or not.
- Compare the performance of Machine Learning and Deep Learning models.
- Analyze evaluation metrics to identify the best-performing model.

## 🛠 Technologies Used

- Python
- TensorFlow
- Keras
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📂 Dataset

**Dataset:** Telecom Customer Churn Dataset

The dataset contains customer demographic information, account details, and service usage used to predict customer churn.

## 🤖 Machine Learning Models

- Logistic Regression
- Decision Tree
- Random Forest
- Support Vector Machine (SVM)
- K-Nearest Neighbors (KNN)
- Naive Bayes
- Multi-Layer Perceptron (MLP)

## 🧠 Deep Learning Models

- LSTM
- Bi-LSTM

## 📊 Evaluation Metrics

The models were evaluated using:

- Accuracy
- Precision
- Recall
- F1-Score
- ROC-AUC Score
- Confusion Matrix

## 📈 Results

- Compared Machine Learning and Deep Learning approaches.
- Achieved high prediction accuracy for customer churn classification.
- Evaluated model performance using multiple classification metrics.
- Identified the best-performing model based on overall evaluation results.
